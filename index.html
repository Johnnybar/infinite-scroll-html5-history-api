HTML5 History API with infinite Scroll and Pagination

Using @TimSeverien's Infinite Scroll + Pagination Experiment, I added a function that reflects the current page in the URL. Came up with a semi-primitive solution but it works:)

May be beneficial for SEO usage on single-page websites, as content is loaded while scrolling but the appearance is single-page. Infinite scrolling is limited to 14 pages, to accomodate the number of sections.
